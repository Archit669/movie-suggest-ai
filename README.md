# Movie Suggest AI

Movie Suggest AI is a web application that recommends movies and TV shows based on user preferences. It utilizes Deno for the backend, Supabase for the database, and vector databases for movie data.

## Tech Stack

### Frontend
- **Framework:** React.js
- **Styling:** Tailwind CSS
- **State Management:** Context API
- **Routing:** React Router
- **Lazy Loading Images:** react-lazy-load-image-component

### Backend
- **Runtime:** Deno
- **Database:** Supabase (PostgreSQL)
- **Web Framework:** Oak

### Other Tools
- **Vector Database:** Used for storing movie data efficiently.
- **API Integration:** Custom API endpoints for fetching movie recommendations.
- **Deployment:** GitHub Pages (Frontend), Heroku (Backend)
- **Version Control:** Git

## Features
- **Recommendation System:** Provides movie and TV show recommendations based on user preferences.
- **Search Functionality:** Users can search for specific movies or TV shows.
- **User Profiles:** Users can create profiles to save their preferences and view personalized recommendations.
- **Responsive Design:** Optimized for various screen sizes and devices.

## Getting Started
1. Clone the repository: `https://github.com/Archit669/movie-suggest-ai>`
2. Install dependencies: `npm install`
3. Start the frontend: `npm run dev`
4. Start the backend: `deno run --allow-net --allow-read server.ts`
5. Access the application in your browser: `http://localhost:5173`
